# Xcuse

A simple web application that generates creative excuses using a custom API. The project demonstrates API integration, dynamic data rendering, and basic JavaScript concepts with a clean and responsive user interface.

## Features

* Generate random excuses
* Search excuses
* Filter by category
* Sort results
* Save favorites (Local Storage)
* Responsive design

## API

[https://excuser-three.vercel.app/v1/excuse](https://excuser-three.vercel.app/v1/excuse)

## Tech Stack

* HTML
* CSS
* JavaScript
* Fetch API

## Setup

1. Clone the repository
   git clone [https://github.com/anaymaheshwari762/Xcuse.git](https://github.com/anaymaheshwari762/Xcuse.git)

2. Open the project folder
   cd Xcuse

3. Run the project
   Open index.html in your browser
